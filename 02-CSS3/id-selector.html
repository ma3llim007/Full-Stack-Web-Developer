ID Selector

The id selector is used the id attribute of an HTML element to select a specific element.

The id of an element should be unique wihtin a page, so the id selector is used to select one unique element.

To select an element with a specific id, write a hash(#) character, followed by the id of the element

Rules

  1. Must Begin with a letter A-Z or a-z
  2. A ID Name Cannot Start With a Number
  3. Must Not Contain any space character
  4. Can be followed by letters(A-Z, a-z), digits (0-9), hyphens("-")and underscores("_")
  5. In Html, All Values are Case-Insensitive
