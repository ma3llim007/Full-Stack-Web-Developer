An Ordered List display a list of item using number, letters in either ascending or descending order

the Ol Tag is used to define an Ordered list; Whereas the li Tag is used to define the items of list.

Ex:-

  1. Windows
  2. Linux
  3. Mac
  4. Android

Types of the Attribute and Value used in Ordered List

Attribute Value
Type 1,A,a,I,i
